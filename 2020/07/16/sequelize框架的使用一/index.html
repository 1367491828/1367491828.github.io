sequelize框架的使用

下载sequelize与对应的数据库模块

npm install sequelize --save

npm install mysql2 --save

npm install moment --save //时间管理模块

配置连接数据库：

定义模型：

定义：

sequelize默认给数据库添加id,createdAt,updatedAt三个字段，id为从1递增的数值，createdAt,updatedAt为datetime数据类型。

timestamps：boolean,当为true时，数据库中会默认设置,createdAt，updatedAt,两个字段，这两个字段为mysql的datetime类型，分别为创建的时间和上次该条数据修改的时间。

freezeTableName: boolean,默认为false,定义时找的就是复数的表名，当为true时，默认mysql的数据库名称就是为define后接的数据库表名.

字段的约束：

如果只有一个约束条件，则可以以键值对的形式定义如，password:Sequelize.STRING.

如果字段有多个约束条件，则需要约束多个条件则需要使用对象的形式，使用对象的形式时，键的取值有：

type: datatypes,约束字段的数据类型，具体类型如下面列出的数据。

allowNull:boolean，该字段是否允许为空。

defaultValue:anytype，该字段的值得类型。

primaryKey:boolean, 该字段是否为主键。

autoIncrement: boolean,该字段值是否为自增。

unique:boolean, 该字段是否为唯一。

validate: {key:value},对数据类型进行检查。

​ validate中的设置：

​ isArray: true, //传入的必须是数组，

​ { isDate: true, //设置为日期

​ isAfter: “2011-11-05”,

​ isBefore: “2011-11-30”

​ }

​ len: [2,10], //字符串长度为2~10

get(){}: this.getDataValue(‘key’)方法可以获取指定字段的值，函数必须返回一个数据，可以对数据处理后，返回数据

set(value){ this.setDataValue(‘key’,value.toUpperCase())}: 首先获得要改变的值，再根据setDataValue方法赋值给某个字段（一般为自己),后接方法，方法可以在外部定义，但是方法必须返回加工后的值，change(value).

datatypes:

Mysql sequelize

VARCHAR(255) STRING

VARCHAR(1234) STRING(1234)

TEXT TEXT

INT INTEGER

BIGINT BIGINT

FLOAT FLOAT

FLOAT(10,2) FLOAT(10,2)

DOUBLE DOUBLE

DOUBLE(10,2) DOUBLE(10,2)

DATETIME DATE

DATE DATEONLY

DATETIME NOW(当前时间)
