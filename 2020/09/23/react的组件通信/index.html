react的组件通信

1.使用props进行通信，在父子组件中使用比较常见，如果在祖孙组件之间传值，需要通过父组件继续将值往下传。

2.消息-订阅模式

npm install pubsub-js --save

import PubSub from ‘pubsub-js’

//消息订阅，一般放在生命周期的componentDidMount阶段

PubSub.subscribe(‘car’, function(msg,data){

​ ……..//msg为消息名不能省略，data为传来的数据

});

//发布消息

PubSub.publish(‘car’, {name: ‘my new car’});

//第一个为消息订阅的标识，第二个为发布的数据
