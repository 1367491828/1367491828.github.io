typescript的学习

1.如果申明了一个变量而没有直接赋值，将会被直接视为any类型。下面将不会报错，默认视为any类型。如果定义的时候没有赋值，不管之后有没有赋值，都会被推断成 any 类型而完全不被类型检查：

var gt;

gt = 23;

gt = 'gfhfh';

2.类型推断：如果没有明确的指定类型，那么 TypeScript 会依照类型推论（Type Inference）的规则推断出一个类型。TypeScript 会在没有明确的指定类型的时候推测出一个类型，这就是类型推论。

var df = '123';

df = 123;

报错：base.ts:6:1 - error TS2322: Type ‘number’ is not assignable to type ‘string’.

6 df = 123;

3.联合类型：使用 ‘ | ’ 管道符来表示或

var unni:string|number; // 默认指定unni只有可能为string和number类型，赋值这两个类型不会报错。

unni = ‘dsfdf’;

unni = true;

报错：base.ts:13:1 - error TS2322: Type ‘boolean’ is not assignable to type ‘string | number’.

13 unni = true;
